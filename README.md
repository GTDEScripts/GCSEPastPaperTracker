# GCSE Past Paper Tracker

An interactive web-based tracker for managing and monitoring your GCSE past paper progress and scores across all subjects.

## Features

✅ **Track Progress** - Mark papers as complete with checkboxes
📊 **Score Tracking** - Enter scores for each paper with automatic percentage calculations
🎯 **Subject Coverage** - Support for all major GCSE subjects:
- Mathematics (3 papers)
- English Language (2 papers)
- English Literature (2 papers)
- Biology (2 papers)
- Chemistry (2 papers)
- Physics (2 papers)
- History (3 papers)
- Geography (3 papers)
- Computer Science (2 papers)

📈 **Analytics** - View:
- Average score percentage
- Best performing session
- Papers completed vs total
- Session totals with percentages

💾 **Data Persistence** - All progress saved locally using browser storage (no account needed)
📥 **Export** - Download your scores as a text file

## How to Use

### Online (GitHub Pages)
Visit: https://gtdescripts.github.io/GCSEPastPaperTracker/

### Local
1. Download `index.html` from this repository
2. Open it in any modern web browser
3. Select a subject from the sidebar
4. For each exam session, enter your score and max marks for each paper
5. Percentages calculate automatically
6. Track your progress with the dashboard at the top
7. Export your scores at any time using the Export button
